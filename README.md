# 🏃 Laufroute Planer

Eine einfache Web-App zum Erstellen von Laufstrecken auf Basis von echten Wegen (keine Luftlinien).

## 🚀 Features

- Karte auf Basis von OpenStreetMap
- Punkte per Klick setzen
- Automatische Routenberechnung entlang von Wegen
- Distanzanzeige in Kilometern
- Undo-Funktion (letzten Punkt entfernen)
- Route zurücksetzen
- Export als JSON-Datei
- Import von gespeicherten Routen

## 🧱 Technologien

- Leaflet (Kartenanzeige)
- OpenStreetMap (Kartendaten)
- OpenRouteService API (Routing)
